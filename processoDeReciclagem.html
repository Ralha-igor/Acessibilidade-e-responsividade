Como começa a recilagem?

A reciclagem do lixo eletrônico começa com uma triagem em que são separados os produtos que ainda podem ser usados ou não.
Em seguida, os aparelhos sem uso são desmontados. A carcaça, a bateria, as placas de circuito e os vidros recebem a destinação
adequada de acordo com os seus componentes.


Como é separado?

Passa por uma triagem onde são separados cada item
Após ser selecionado, os que nao são usados eles serão desmontados.

Destinação Final

Com todas as obras-primas separadas, é a hora de destinar as industrias, para que seja feito um novo equipamento eletronico para o mercado consumista. Desta fomra
as empresas diminuem a extração de recursos naturais e preserva o nosso meio ambiente.

Basicamente os resíduos eletrônicos são coletados para fazer uma logistica reversa, sendo reutilizados para novos componentes eletronicos se tornando uma fonte inesgotaval de recursos.
